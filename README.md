## PythonRestApiTemp
Python(Flask + MySQL + SQLAlchemy)

## Check
1. mysql installed
2. virtualenv installed
3. python(ver 2.7.5) installed

## Set Project Environment
1. clone project
2. move to project folder and set virtualenv and virtualenv activate
 * >  virtualenv venv
 * > . venv/bin/activate
3. install package
 * > pip install -r requirements.txt
